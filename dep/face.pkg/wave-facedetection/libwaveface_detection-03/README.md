# 人脸检测 v1.2.1
 人脸检测算法适用于动态人脸识别项目，人脸检测的阈值为0.7最佳。
## 发布日期
 2018-08-07
## 平台
centos 64
## 修改内容
添加GPU id 接口，编译GPU 资源管理</br>
添加初始化锁，避免多线程初始化崩溃</br>
## 依赖第三方库
libwavedl_centos64_gpu_v0.1.1.so
libwavelog_v0.0.0.so
libgpu_manager.so
## 调用模型名字
face_detection_centos64_v1.2.1.wave
